# FitTrack

A minimalist iOS app for tracking gym workouts.

## Features

- 3-day workout split (Chest/Shoulders/Triceps, Back/Biceps, Legs/Abs)
- Track sets, reps, and weights for each exercise
- Rest timer between sets
- Workout history and statistics
- Push notifications for workout reminders (Mon/Wed/Fri)

## Tech Stack

- SwiftUI
- SwiftData

## Requirements

- iOS 17.0+
- Xcode 15.0+
